 HTML的标签： 
标签有两种：起始标签和空标签
起始标签，开始标签 <标签名> 和 结束标签 </标签名>
空标签只有开始标签
标签的属性只存在于起始标签里
标签的属性：id=“ ”标识符，是唯一的
style=“ ”样式属性
class=“ ”类名属性
name=“ ”名称属性（类似名字）
